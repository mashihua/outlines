h1.HTML5 Outlines 

This is a tool implement the creating an html5 outlines algorithm 

h2.Instructions

h3.For use in browser:
<p>just load the script onto a server of your choice and add a bookmarklet with the following code:</p>
<pre>javascript:(function(){var s = document.createElement('script'),head = document.getElementsByTagName('head')&#91;0&#93;;head.appendChild(s);s.src='https://raw.github.com/mashihua/outlines/master/js/bookmarklet.js';}());
</pre>

h3.For use in Node.js:
<pre>git clone https://mashihua@github.com/mashihua/outlines.git
npm link

var HTML5 = require(‘html5’),
    jsdom = require(‘jsdom’);
    window = jsdom.jsdom(null, null, {parser: HTML5}).createWindow(),
    parser = new HTML5.Parser({document: window.document}),
    outlines;
parser.parse('&lt;body&gt;Foo&lt;/body&gt;');
</pre>
